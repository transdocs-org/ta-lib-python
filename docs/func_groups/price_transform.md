# 价格变换函数
### AVGPRICE - 平均价格
```python
real = AVGPRICE(open, high, low, close)
```

### MEDPRICE - 中位数价格
```python
real = MEDPRICE(high, low)
```

### TYPPRICE - 典型价格
```python
real = TYPPRICE(high, low, close)
```

### WCLPRICE - 加权收盘价
```python
real = WCLPRICE(high, low, close)
```
