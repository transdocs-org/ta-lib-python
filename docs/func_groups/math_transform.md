# 数学变换函数
### ACOS - 向量三角反余弦函数
```python
real = ACOS(real)
```

### ASIN - 向量三角反正弦函数
```python
real = ASIN(real)
```

### ATAN - 向量三角反正切函数
```python
real = ATAN(real)
```

### CEIL - 向量向上取整函数
```python
real = CEIL(real)
```

### COS - 向量三角余弦函数
```python
real = COS(real)
```

### COSH - 向量双曲余弦函数
```python
real = COSH(real)
```

### EXP - 向量指数函数
```python
real = EXP(real)
```

### FLOOR - 向量向下取整函数
```python
real = FLOOR(real)
```

### LN - 向量自然对数函数
```python
real = LN(real)
```

### LOG10 - 向量以10为底的对数函数
```python
real = LOG10(real)
```

### SIN - 向量三角正弦函数
```python
real = SIN(real)
```

### SINH - 向量双曲正弦函数
```python
real = SINH(real)
```

### SQRT - 向量平方根函数
```python
real = SQRT(real)
```

### TAN - 向量三角正切函数
```python
real = TAN(real)
```

### TANH - 向量双曲正切函数
```python
real = TANH(real)
```
